# Release Notes - Odaai POS v1.0.0

**Release Date:** March 3, 2026

## 🎉 Major Release - Production Ready

This is the first production release of Odaai POS, a comprehensive Point of Sale system designed for restaurants and hospitality businesses. The system now includes all core functionality needed for daily operations.

---

## 🚀 Highlights

### Restaurant Operations
- **Split Bill & Table Merge** - Flexible billing for group dining scenarios
- **Floor Plan Management** - Visual table layout designer with 7 status colors
- **Table Movement** - Seamlessly transfer customers between tables
- **Open Tab System** - Allow customers to keep tabs open and pay later
- **Bill Request** - Waiters can initiate payment requests from tables

### Kitchen Management
- **Enhanced KDS** - Real-time stats badges showing today's order metrics
- **Order Type Badges** - Clear visual indicators for dine-in, delivery, and takeout
- **Auto-sync Table Status** - Kitchen operations automatically update table states

### Delivery & Customer Service
- **Phone Delivery** - Dedicated delivery order form with customer information
- **Auto-lookup** - Automatic customer retrieval by phone number
- **Customer Management** - Comprehensive customer database with order history

### Product Management
- **Category System** - Hierarchical product categorization
- **Tax Management** - Support for multiple tax rates and configurations
- **Product Modifiers** - Customizable menu items with add-ons and options

### Staff & Administration
- **Attendance GPS** - Location-based employee check-in/check-out
- **Loyalty Dashboard** - Quick actions for points adjustment and tier management
- **Settings Persistence** - Store configurations saved locally
- **Backup & Restore** - Full data backup and recovery functionality

### Multi-currency & i18n
- **Exchange Rates** - Database-integrated currency conversion
- **4 Languages** - English, Vietnamese, Thai, and Korean support
