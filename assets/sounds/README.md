# KDS Audio Assets

이 폴더에는 주방 디스플레이 시스템(KDS)에서 사용되는 오디오 알림음 파일들을 저장합니다.

## 필요한 파일

다음 3개의 MP3 파일을 이 폴더에 추가해야 합니다:

1. **new_order.mp3** - 신규 주문 접수 시 재생
   - 권장: 짧고 명확한 알림음 (1-2초)
   - 예: "띵동" 또는 벨소리

2. **order_ready.mp3** - 조리 완료 시 재생
   - 권장: 긍정적인 완료 사운드 (1-2초)
   - 예: "땡" 또는 성공 알림음

3. **urgent_order.mp3** - 긴급 주문 표시 시 재생
   - 권장: 주의를 끄는 강한 알림음 (1-3초)
   - 예: 경고음 또는 긴급 알림

## 오디오 파일 준비 방법

### 옵션 1: 무료 음원 사이트 활용
- Freesound.org
- Zapsplat.com
- Mixkit.co

### 옵션 2: 직접 녹음
- 스마트폰 녹음 앱 사용
- Audacity 등의 무료 편집 도구로 편집

### 옵션 3: 온라인 TTS 활용
- "새 주문이 접수되었습니다" 등의 음성 메시지 생성
- 한국어 TTS 서비스 이용

## 파일 형식 요구사항
- 형식: MP3 (권장)
- 길이: 1~3초 이내
- 파일명: 위에 명시된 정확한 이름 사용

## 테스트

오디오 파일 추가 후:
```bash
flutter run
```

KDS 화면에서 다음을 테스트:
1. 신규 주문 생성 → new_order.mp3 재생 확인
2. 조리 완료 버튼 → order_ready.mp3 재생 확인
3. 긴급 주문 표시 → urgent_order.mp3 재생 확인

## 주의사항
- 음량이 너무 크거나 작지 않도록 조절
- 주방 환경(시끄러운 환경)을 고려한 명확한 사운드 선택
- 파일명과 경로가 정확히 일치해야 함
